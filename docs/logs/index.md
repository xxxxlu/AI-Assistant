
# 日志中心

## 作用

记录系统运行中的关键事件、错误、安全访问，确保可追溯。

## 日志类型

| 文件 | 用途 | 更新频率 |
|------|------|----------|
| `errors.md` | 错误与异常 | 实时 |
| `security.md` | 安全相关访问 | 实时 |
| `file-access.md` | 敏感文件访问 | 实时 |
| `memory-gc.md` | 记忆清理记录 | 定期（GC 触发） |

## 格式规范

每条日志包含:

```markdown
## YYYY-MM-DD HH:mm:ss

- **类型**: Error | Warning | Info | Security
- **来源**: <模块/文件>
- **描述**: <详细信息>
- **处理**: <已采取的行动>
```

## 保留策略

- 错误日志: 保留 90 天
- 安全日志: 保留 180 天
- 常规日志: 保留 30 天

## 清理命令

```bash
# 清理 90 天前的日志
pnpm memory:gc -- 90
```
