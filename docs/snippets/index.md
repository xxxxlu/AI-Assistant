# Snippets 经验碎片库

## 用途

记录短周期、高价值、可复用的执行片段。

## 命名建议

- 路径：`docs/snippets/YYYY-MM/<slug>.md`
- 只写“场景 + 动作 + 结果 + 复用条件”

## 模板

- 参考：`/snippets/template`

## 写入原则

1. 一条 snippet 只解决一个问题。
2. 长文经验请升级到 `retrospectives`。
3. 涉及敏感信息时，仅写脱敏结论。
